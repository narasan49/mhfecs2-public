このサイトについて

ここはモンスターハンターフロンティア(MHF)の装備をMHFユーザー同士で共有するためのサイトです。 それぞれのMHFユーザーが考えた装備の組み合わせを投稿してもらうことでデータベース化し、自由に閲覧できるようにします。 会員登録することで投稿履歴を確認できたり、他の人の投稿を保存することができるようになります。 ご意見・ご要望ございましたらお気軽に下記の宛先にご連絡ください。 不具合を発見された場合も報告してくださると助かります。 当サイトはリンクフリーです。

E-mail: narasan49_at_gmail.com, Twitter: naraMHF

装備の保存場所として利用するもよし、 自慢の装備をお披露目するもよし、 初心者のための装備を投稿するもよし

使い方

閲覧編

  1. 絞り込み検索
    2. 武器種・スキル・タグに条件をつけることで、絞り込み検索ができます。
  2. 検索結果
    3. 検索条件にあった装備が表示されます。使用した装備品、装飾品、スキルカフと発動スキル、投稿者のコメントが確認できます。
    • :いいねボタン
    • :保存ボタン。会員登録することで利用できます。
    • :クリップボタン。装備データをクリップボードにコピーします。

投稿編・クリップ貼付

ご自身が組んだ装備を実際に投稿してこのサイトで公開する方法を2通りご紹介します。1つ目は、クリップボードに保存したMHSX2Gでのシミュ結果を貼り付けることで投稿する方法です。

  1. 名前入力
    2. 検索時に表示される名前を入力することができます。ログイン時は自動的に入力されます。
  2. 武器種の選択
    3. 必須項目。武器種を選択します。14武器種の他に剣士汎用、ガンナー汎用という項目もあります。
  3. クリップ貼付
    4. クリップボードに保存した装備情報をそのまま整形等せずに貼り付けてください。
  4. コメントの記入
    5. 装備の説明を自由に記入してください。400字以内。
  5. タグの選択
    6. 用途・装備の特徴にあったタグを選択してください。検索で見つけやすくなります。追加してほしいタグなどの要望も受け付けています。検討の上、追加します。
  6. 投稿確認・投稿
    7. 投稿内容の確認ボタンで発動スキル等をチェックします。問題なければ投稿して、操作完了です。リセットボタンで入力内容がクリアされます。

投稿編・直接入力

発動スキルから武具の候補を検索し、その中から選択することで投稿できます。

  1. スキル種別の選択
    2. スキル種別を選択します。選択後、2番のスキルが表示されます。
  2. スキルの選択
    3. 選択したスキルを含む防具を検索します。
  3. 辿異スキルの選択
    4. 選択されなかった辿異スキルを含む防具を除外します。
  4. その他条件の選択
    5. 防具の種別や強化状況について検索範囲を指定します。

検索結果の一例です。

  1. 武具の検索結果
    2. 武具の検索結果を表示します。タブによって部位の切り替えができます。スライドバーを動かすことで、武具に含まれる発動スキルの数が指定できます。
  2. カフの検索結果
    3. カフの検索結果を表示します。選択されたカフは下部に表示され、その個数を選択できます。装飾品についても同様です。
